## Project Summary

LineGuard makes the stack fit concrete by starting with an old manufacturing manual and handwritten form, then turning it into a Dozuki-ready workflow: visual guides, live publishing, training progress, worker feedback, React/Next.js UI work, NestJS-style API boundaries, SQL-backed audit data, legacy PHP compatibility, AWS-style debugging, and AI review guardrails.

## Core Fit

The project sits where Dozuki's role sits: product engineering, manufacturing process quality, data integrity, documentation, cloud debugging, and AI-assisted development with human review.
